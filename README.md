## 📖 Project Overview
This project was created as part of the Digital Literacy course. The aim of this project is to understand and apply essential digital skills such as using online platforms, creating digital content, communicating professionally, and staying safe on the internet. 

The project includes five tasks that focus on different aspects of digital literacy, including infographic creation, building a digital portfolio, exploring coding platforms, learning email etiquette, and understanding cybercrime awareness.

---

## 📂 Project Structure

- `task-4-email-etiquette/` → Email drafts and social media checklist

## 📝 Task Summary

### 🌐 Task 2: Digital Portfolio

### 💻 Task 3: Platforms
Solved a beginner problem on HackerRank and created a Google Form quiz on digital literacy.

### 📧 Task 4: Email Etiquette

### 🔐 Task 5: Cybercrime Awareness
